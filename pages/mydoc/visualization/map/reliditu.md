 热力地图以特殊高亮的形式显示数据和地理区域的关系。 

![image-20191126155138761](../../../../images/image-20191126155138761.png)


## 数据

| 数据 | 描述         |
| ---- | ------------ |
| 经度 | 经度信息字段 |
| 纬度 | 纬度信息字段 |
| 度量 | 度量字段     |
